This mod contains the Butcher animation from Heroes of the Storm for BGEE/SOD/BG2EE.
There is no playable content so this is intended to be a modding resource only. (A mod using the animation is in the works however.)

The mod contains the animation .bam files as well as some sound files from the Butcher in Heroes of the Storm.
